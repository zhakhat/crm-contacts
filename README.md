Что мы сделали в целом:
CRUD:
Create (создать)
Read (прочитать)
Update (обновить)
Delete (удалить)

Пользователь видит список контактов (таблица).
Может искать по имени/почте и фильтровать по статусу.
Нажимает “Add Contact” → открывается форма создания.
Нажимает “Edit” → открывается форма редактирования уже существующего.
Нажимает “Save” → данные уходят в API, список обновляется.
Нажимает “Delete” → контакт удаляется.

(CRUD через axios) приложение хранит контакты не в памяти, но на сервере (json-server)
db.json - база данных
src/api - всё, что связано с HTTP
src/components - UI-компоненты
src/types - типы данных

Агоритм шагов:
1) создали базу данных
2) добавили в main bootstrap 
3) создали типы и описали 
4) создали ui для филтров, таблиц и форм 
5) и создали api
6) потом залили все это в git
